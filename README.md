# Flood Susceptibility Modelling using ML and CMIP6

This repository contains the core scripts used in the study:

"Climate-Driven Explainable Machine Learning Framework Integrating CMIP6 Projections for Future Flood Susceptibility"

## Contents
- Data preprocessing
- Bias correction (CMIP6)
- Machine learning models
- Flood susceptibility prediction

## Requirements
Python 3.9
Libraries: scikit-learn, xgboost, shap, numpy, pandas

## Note
Data are from public sources (CMIP6, IMD, Sentinel-2).
